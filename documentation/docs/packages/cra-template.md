# cra-template
![npm latest package](https://img.shields.io/npm/v/@monkvision/cra-template/latest.svg)

## Overview
This package exports multiple Create-React-App temapltes that allow developers to create applications that implement our
SDK in a single command line. As a developer, you don't have to install this package. You can simply mention the name
of the template you want to use when using the CRA CLI in your terminal.

## Complete Documentation
As every other package in the SDK, please refer to
[its README file](https://github.com/monkvision/monkjs/blob/main/packages/cra-template/README.md) for a complete
documentation on how this package works.
